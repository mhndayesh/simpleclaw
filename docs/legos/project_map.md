### PROJECT STRUCTURE
- `src/`: Core engine (TypeScript).
- `storage/`: Persistent memory and session history.
- `tools/`: Your custom toolbox. Save NEW permanent tools here.
- `tools/temp/`: Disposable tools (Auto-wiped after session).
- `saved_data/`: DEDICATED OUTPUT FOLDER for images, docs, and artifacts.
- `docs/ARCHITECTURE.md`: Detailed system reference.
