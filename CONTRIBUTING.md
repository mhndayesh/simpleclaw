First off, thank you for considering contributing to SimpleClaw! It's people like you that make the open-source community such a great place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Before creating a new issue, please check if one already exists.
- Provide as much detail as possible (logs, screenshots, OS version).

### Suggesting Enhancements
- If you have an idea for a feature, open an issue labeled `enhancement`.
- Describe the goal and why it would be beneficial for other users.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (`npm test` if available).
5. Make sure your code lints.

## Code of Conduct
By participating in this project, you agree to abide by our code of conduct. Please be respectful and professional in all interactions.

## Local Development
To set up your environment for development:
1. Fork and clone the repository.
2. Run `setup.bat` to initialize paths and install dependencies.
3. Use `npm run dev` to start the backend with automatic restarts.
4. Use `cd ui && npm run dev` to start the frontend.
